# Программа с окружностью

Программа на C# Windows Forms, которая отображает окружность в центре окна с возможностью изменения радиуса.

## Функциональность

- Отображение окружности с радиусом 100 в центре окна программы
- Управление радиусом с помощью клавиш:
  - **Стрелка вправо (→)** - увеличение радиуса
  - **Стрелка влево (←)** - уменьшение радиуса
- Ограничения:
  - Минимальный радиус: 1 пиксель
  - Максимальный радиус: половина ширины окна
- Отображение текущего значения радиуса в левом верхнем углу

## Технические особенности

- Окружность всегда остается в центре окна
- При изменении размера окна автоматически пересчитывается максимальный радиус
- Плавная отрисовка с антиалиасингом
- Двойная буферизация для предотвращения мерцания

## Компиляция и запуск

### Требования
- .NET 6.0 или выше
- Windows (для Windows Forms)

### Команды для компиляции и запуска

```bash
# Компиляция
dotnet build CircleApp.csproj

# Запуск
dotnet run --project CircleApp.csproj
```

### Альтернативный способ (только компиляция в .exe)
```bash
dotnet publish -c Release -o ./publish
```

После этого в папке `./publish` появится исполняемый файл `CircleApp.exe`.

## Принцип работы

Координаты окружности задаются через прямоугольник, в который она вписана:
- Центр окна: `(width/2, height/2)`
- Левый верхний угол прямоугольника: `(centerX - radius, centerY - radius)`
- Размер прямоугольника: `(2 * radius, 2 * radius)`

При изменении радиуса происходит пересчет координат прямоугольника, что приводит к изменению размера окружности при сохранении ее центрального положения.
